Preciso revisar e completar o documento de uma tela do FitnessApp.

**Arquivo:** $ARGUMENTS

Faca o seguinte:

1. Leia o documento da tela indicada
2. Leia o template em `docs/00-template-tela.md` para comparar
3. Identifique:
   - Campos marcados com `⚠️ PENDENTE`
   - Secoes vazias ou com conteudo generico
   - Secoes do template que estao faltando no documento
4. Apresente um **resumo numerado** do que esta faltando
5. Para cada item, me faca perguntas **objetivas com opcoes** (ex: "Sim/Nao", "Opcao A / Opcao B / Outro")
6. Aguarde minhas respostas
7. Atualize o documento com as informacoes coletadas
8. Atualize o status:
   - Se TUDO preenchido → 🟢 CONCLUIDO
   - Se ainda tem pendencias → 🟠 PENDENTE REVISAO
9. Atualize o indice em `docs/01-visao-geral.md` com o novo status
