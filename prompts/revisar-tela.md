# Prompt: Revisar e Completar Tela

> **Como usar:** Use quando uma tela já foi criada mas está incompleta ou marcada com ⚠️ PENDENTE.
> Ideal para sessões de revisão com o cliente.

---

```
Preciso revisar e completar o documento da seguinte tela do FitnessApp:

**Arquivo:** [CAMINHO DO ARQUIVO, ex: docs/screens/auth/login.md]

Por favor:

1. Leia o documento atual desta tela
2. Identifique todos os campos marcados com "⚠️ PENDENTE" ou que estejam vazios
3. Me apresente um resumo do que está faltando, organizado por seção
4. Para cada item pendente, me faça as perguntas necessárias para preenchê-lo
5. Após minhas respostas, atualize o documento com as informações coletadas
6. No final, atualize o status da tela para 🟢 CONCLUÍDO se tudo estiver preenchido, ou 🟠 PENDENTE REVISÃO se ainda houver itens abertos

Seja específico nas perguntas — prefiro perguntas objetivas com opções quando possível.
Por exemplo: "O botão de login deve estar desabilitado até o formulário ser válido? (Sim/Não)"
```
